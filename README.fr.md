## À propos de moi

Je suis un **développeur applicatif** basé en France, spécialisé dans le développement web backend avec **PHP** et **Symfony**.  
Après une reconversion professionnelle, j'ai obtenu mon titre professionnel de développeur web et web mobile et je travaille actuellement en alternance en tant que concepteur-développeur d'applications chez [MTFibertech](https://mtfibertech.fr).
